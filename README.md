# CSC755M Project 1: Sorting algorithms visualization

An interactive Streamlit-based application that visualizes and empirically compares the growth rates of classic and Shell-based sorting algorithms using swap and comparison counts across varying input sizes.

## How to run:

1. Clone the repository.
2. install the requirements `pip install -r requirements.txt`
3. Run the app. `streamlit run app.py` or `python -m streamlit run app.py`
